## Development history (private repo only)

The private development repo additionally carries: `CHANGELOG.md` (Keep-a-Changelog release notes), `VERSIONS.md` (every dependency's blessed version), `UPGRADE.md`, `docs/research/` (research fit-analyses behind each design), `docs/superpowers/plans/` (per-release build plans), `docs/audit/`, `eval/` (the faithfulness/retrieval eval harnesses), and historical runbooks/resume notes. Those are working documents, excluded from the public mirror by design; the docs above are the maintained current-state set.

> **Two-repo model**: the private repo is the source of truth; the public repo is a scrubbed mirror regenerated by a gated pipeline — never hand-edited. If you're reading this in the public repo and something looks missing, it's one of the private working documents above.
